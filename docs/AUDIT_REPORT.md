## 체크리스트 (3-Layer)

### A. Correctness — 정답성

| ID | 의미 |
|---|---|
| A1 | 표시된 정답이 실제로 옳다 |
| A2 | 다른 모든 보기가 명백한 오답이다 |
| A3 | 출제 의도 (출처 명세 / ADSP·SQLD 시험 표준) 와 일치 |
| A4 | 외부 컨텍스트 (그림·표·코드) 없이 문항 본문만으로 풀이 가능 |
| A5 | 정답이 한 개로 유일 (애매성·복수 정답 없음) |

### B. Distractor Quality — 오답 보기

| ID | 의미 |
|---|---|
| B1 | 그럴듯한 함정 (학습자가 빠지기 쉬운 오개념 반영) |
| B2 | 정답·오답 보기 길이 균형 (M4: 1.6× 이상 차이 X) |
| B3 | 보기 간 의미 중복 없음 |
| B4 | 정답 키워드가 보기 자체에 노출 안 됨 (단서 누설) |
| B5 | 보기 형식·문법 통일 (모두 명사절·문장형 등 일관) |

### C. Explanation Quality — 해설

| ID | 의미 |
|---|---|
| C1 | why_correct (왜 정답인가) 한 문장 이상 명확 |
| C2 | why_wrong (다른 보기가 왜 오답인지) 보강 |
| C3 | core_concept (핵심 개념) 한 문장 요약 포함 |
| C4 | 학습 팁 또는 암기법 (mnemonic) 언급 |
| C5 | related (관련 개념·다음 학습 링크) 언급 |
